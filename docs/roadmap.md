# Roadmap

## Phase 1

- compile a tiny runtime
- define core data structures
- initialize structured logging
- support one in-memory thread store

## Phase 2

- add local host tools
- unify workspace-relative path behavior
- add robust tool validation and typed errors

## Phase 3

- implement one real channel end-to-end
- persist cursors and thread history
- add a debugging timeline view

## Phase 4

- add model-provider adapters
- add prompt assembly and tool-loop execution
- add replayable failure traces
